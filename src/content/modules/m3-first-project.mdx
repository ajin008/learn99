# Build Your First Project

### Goal

Create a **personal portfolio website** in 15 minutes.

---

### Steps

1. Prompt AI → “Build me a personal portfolio site with my name, photo, and 3 projects.”
2. Copy code → Paste into **Replit** or local editor.
3. Deploy → Push to **Vercel (free hosting)**.
4. Share → Send your live link to friends instantly.
