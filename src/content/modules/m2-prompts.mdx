# Mastering Prompts for Coding

### Writing AI Prompts

- Be specific: “Build me a landing page with a headline and button.”
- Use context: “Make it mobile-friendly and modern.”
- Give format: “Return only HTML/CSS code.”

---

### Prompt Patterns

1. **Explain** → Ask AI to explain a concept in simple words.
2. **Generate** → Tell AI to create code.
3. **Fix** → Paste broken code and ask AI to debug.
4. **Deploy** → Ask how to publish online.
